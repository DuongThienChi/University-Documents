Lectures

This page displays what we have covered each week, with slides, lecture code, and relevant links. NOTE: Lectures will not be video recorded; sorry for the inconvenience.

Week 10 ()

This week we will talk about creating and querying databases in an Android app.

  • slides: databases (PDF)

Week 9 ()

This week we will talk about services and notifications.

  • slides: services, notifications (PDF)
  • lecture code (ZIP): Jukebox, Downloader
    • Jukebox: JukeboxActivity.java, activity_jukebox.xml, JukeboxService.java
    • Downloader: DownloaderActivity.java, activity_downloader.xml, DownloaderService.java, Downloader.java, AndroidManifest.xml

Week 8 ()

This week we will talk about maps and location-based apps. We will also talk about "swiping" and finger/mouse gestures.

  • slides: maps, swiping (PDF)
  • lecture code (ZIP): SwipeDemo, CityFinder
    • SwipeDemo: OnSwipeListener.java, SwipeDemoActivity.java
    • CityFinder: AndroidManifest.xml, CityFinderActivity.java, activity_city_finder.xml
  • Links: GoogleMap, MapFragment

Week 7 ()

This week we will talk about basic 2D graphics and animation, such as for making a simple game.

  • slides: 2D graphics (PDF)
  • lecture code (ZIP): Targets, BouncingBall
    • Targets: ExampleView.java, TargetsView.java
    • BouncingBall: BouncingBallView.java, DrawingThread.java, Sprite.java

Week 6 ()

This week we will talk about: text-to-speech and speech-to-text in an app; lists and more advanced GUI widgets; designing for different device sizes and features; and fragments.

  • slides: text to speech, lists, fragments (PDF)
  • lecture code (ZIP): InsultList, FragmentFun
    • InsultList: InsultListActivity.java, activity_insult_list.xml phrases.txt
    • FragmentFun: MainFragment.java, fragment_main.xml, MainActivity.java, activity_main.xml, activity_main.xml (landscape), DetailsFragment.java, fragment_details.xml, DetailsActivity.java, activity_details.xml
  • assigned: homework assignment 4

Week 5 ()

We canceled class this week because the instructor was sick.

Week 4 ()

This week we will talk about activity state and Bundles, as well as application-level preferences. We will also learn how to make an app that interacts with data files, and learn how to capture photos using the Android device's camera.

  • slides: camera, state/preferences, files/storage (PDF)
  • lecture code (ZIP); MainActivity.java, activity_main.xml, DetailsActivity.java, activity_details.xml, dict.txt, AndroidManifest.xml (TMNT image chooser v3)

Week 3 ()

This week we will talk about the lifecycle of an activity in Android as it is created, paused, destroyed, etc. We will learn how to create an app with more than one activity and launch one activity from another using Intents. We also learned about how to play music and audio clips using the MediaPlayer class.

  • slides: MediaPlayer, activity lifecycle, intents (PDF)
  • lecture code (ZIP); MainActivity.java, activity_main.xml, DetailsActivity.java, activity_details.xml (TMNT image chooser v2)
  • youtube-dl and ffmpeg (software for downloading/converting media)

Week 2 ()

This week we will cover how to lay out a GUI in Android, as well as much more about many GUI widgets and events that they can handle. We will also briefly touch on the Android action bar.

  • slides: GUI widgets, layout, action bar (PDF)
  • lecture code (ZIP); MainActivity.java, activity_main.xml (TMNT image chooser)
  • assigned: homework assignment 2
  • related link: GIMP free image editor

Week 1 ()

This week we will cover some general information about the course, background information about the Android platform, and learn enough basics to create our first simple app.

  • slides: creating a simple app; Java review (PDF)
  • lecture code (ZIP); MainActivity.java, activity_main.xml (Berkeley number guessing game)
  • assigned: homework assignment 1
